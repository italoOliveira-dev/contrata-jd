## Rode o projeto

- No seu terminal: <br>
  <ol>
    <li>git clone https://github.com/italoOliveira-dev/contrata-jd.git</li>
    <li>cd contrata-jd</li>
    <li>npm install</li>
    <li>npm run dev</li>
  </ol>
- Abra no navegador: `http://localhost:5173`
